## INSTALATION WITH `gcc`
```bash

git clone https://github.com/mitsuki31/suzumi-airlines.git

## -- Change directory
cd suzumi-airlines

## -- Compile file
g++ suzu_air.cpp -o suzu_air

## -- Run program
chmod +x suzu_air
./suzu_air
```
